# HTML (HyperText Markup Language)

**HTML (HyperText Markup Language)** adalah bahasa markup standar yang digunakan untuk membuat **struktur dasar dan konten halaman website**. HTML menggunakan kumpulan **tag** untuk memberi tahu web browser bagaimana menampilkan teks, gambar, video, tautan, dan elemen multimedia lainnya.
