HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dasar dan konten halaman website. HTML menggunakan rangkaian tag, seperti <h1> (judul) atau <p> (paragraf), untuk memberi tahu web browser cara menampilkan teks, gambar, dan multimedia lainnya. HTML adalah kerangka utama yang mendasari hampir seluruh situs web di internet.
Berikut adalah poin-poin penting mengenai HTML:

- Fungsi: Menentukan struktur halaman web, membuat elemen seperti tautan (hyperlink), paragraf, judul, tabel, dan menyisipkan gambar/video.
- Cara Kerja: Dokumen HTML ditulis menggunakan editor teks (seperti VS Code) dengan ekstensi .html, lalu diinterpretasikan oleh browser (Chrome, Firefox, dll.) untuk ditampilkan sebagai halaman web.
  Struktur Dasar: Terdiri dari elemen <head> (informasi meta) dan <body> (konten yang terlihat).
- Tag: Kode penanda yang diapit kurung siku, contoh: <title> untuk judul, <p> untuk paragraf, <a> untuk tautan.
- Sejarah: Diciptakan oleh Sir Tim Berners-Lee pada tahun 1991. Saat ini, versi yang umum digunakan adalah HTML5.

HTML bukan bahasa pemrograman (karena tidak memiliki logika seperti if-else), melainkan bahasa penanda konten. Untuk membuat website menarik, HTML umumnya dikombinasikan dengan CSS (untuk desain/tampilan) dan JavaScript (untuk interaktivitas).
